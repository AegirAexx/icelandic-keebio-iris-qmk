# Iris rev.7 & Iris CE rev.1

Custom QMK keymaps for Keebio Iris split keyboards with Icelandic layout.

## Overview

My personal keymaps for the Keebio Iris, designed for coding on Linux and Windows. The symbol layers put all programming characters within easy reach, minimizing the hand gymnastics typically needed for brackets, braces, and other symbols.

## Keymap Cheat Sheet

See [KEYMAP.md](KEYMAP.md) for a visual layout reference (printable on A4).

## Features

- **Icelandic QWERTY** - Base layer with Icelandic characters (Ð, Þ, Æ, Ö, ´)
- **Developer-friendly symbols** - All coding symbols (brackets, braces, operators) accessible via LMOD/RMOD layers without awkward finger stretches
- **Tri-layer navigation** - Hold both LMOD + RMOD to access arrows, media controls, and a numrow
- **Combos** - Left Shift + Right Shift for Caps Lock
- **Mod-tap Enter** - Tap for Enter, hold for Alt

## Layers

| Layer | Activation | Purpose |
|-------|------------|---------|
| QWERTY | Default | Icelandic base layout |
| SYMB_L | Hold LMOD | Symbols and F-keys |
| SYMB_R | Hold RMOD | Symbols and F-keys (mirrored) |
| NAV | Hold LMOD + RMOD | Navigation, media, numrow |
